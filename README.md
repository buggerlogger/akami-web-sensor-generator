# akami-web-sensor-generator
akami sensor web plugin openbullet

{version};{state};{f1};{f2};{relTimestamp};{hwMarker};{flags};{encryptedBody}

how to use it 

you should run it with openbullet tool by add this plugin in openbullet https://github.com/openbullet/OpenBullet2/releases/tag/0.3.2

how to add 

copy AkamaiWeb.dll and add in UserData\Plugins in opebullet folder

and run SensorDataGenerator.exe

and open openbullet
